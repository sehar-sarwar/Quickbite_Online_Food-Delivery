# Quickbite_Online_Food-Delivery

## About
QuickBite is a responsive food ordering web application that allows users to browse menu items, search and filter products, manage their cart, and place orders online. It is built using HTML, CSS, JavaScript, PHP, and MySQL, featuring dark mode, dynamic cart management, promo codes, and an interactive user interface.
## Features
-🔍 Real-Time Search & Filtering
-🛒 Smart Cart Management
-🌙 Dark Mode Support
-📦 Online Order Placement


## Technologies Used
HTML | CSS | JavaScript | PHP | MySQL

## How to Run
1. Clone this repo
2. Import db.sql in phpMyAdmin
3. Open index.html in browser
